## Installation
___

```
sudo npm install
```

```
sudo npm install kill-port
```

```
sudo chmod 755 say.sh
```

```
sudo npm install @abandonware/bluetooth-hci-socket --unsafe-perm
```

## Start
___

- In order to launch the script, type :

```
sudo sh start.sh
```

- Then wait few seconds

## Stop
___

- If you want to stop it, just type :

```
sudo sh kill.sh
```

Enjoy :)
